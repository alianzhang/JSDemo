## Browser-Sync

https://www.browsersync.io/

安装：`npm install -g browser-sync`

启动：`browser-sync start --server --files "css/*.css, js/*.js, *.html"`
    
  该命令表示启动一个服务器同时监视指定文件的变化，当文件发生变化，自动同步浏览器刷新。
